📚 **Documinds Enterprise**
Documinds Enterprise is a Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents and have real-time conversations with their data. Powered by LangChain, Hugging Face, and ChromaDB, this tool turns static documents into interactive knowledge bases.

Meet **Jassi**—your dedicated AI assistant who strictly uses your uploaded documents to provide accurate, context-aware answers.

🚀 **Features**
1. PDF Intelligence: Upload any PDF and the system automatically chunks, embeds, and indexes the content.
2. Persistent Vector Store: Uses ChromaDB to store document embeddings for lightning-fast retrieval.
3. State-of-the-Art LLM: Powered by the Qwen2.5-7B-Instruct model via Hugging Face.
4. Custom Personality: Integrated AI assistant named Jassi with a consistent persona.
5. Web Interface: Built with Flask for a smooth, user-friendly experience.

🛠️ **Tech Stack**
1.  Framework: Flask
2. AI Orchestration: LangChain
3. Embeddings: sentence-transformers/all-mpnet-base-v2
4. LLM: Qwen/Qwen2.5-7B-Instruct (Hugging Face)
5. Vector Database: ChromaDB
6. Frontend: HTML/CSS (Jinja2 Templates)
